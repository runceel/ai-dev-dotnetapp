# イベント参加登録システム 仕様書

## 概要

イベントを作成し、参加者が登録できるイベント参加登録システムを構築する。

## 機能一覧

### Events モジュール

- イベントの作成（イベント名、開催日時、定員、説明）
- イベント一覧の表示（開催日時の降順）
- イベント詳細の表示

### Registration モジュール

- イベントへの参加登録（参加者名、メールアドレス）
- 定員以内なら参加確定、定員を超えた場合はキャンセル待ちとして登録
- 参加キャンセル機能（キャンセル時にキャンセル待ち先頭の人を自動で繰り上げ）
- イベントごとの参加者一覧の表示

## 画面構成（Blazor Server）

| 画面 | 説明 |
|-----|------|
| イベント一覧 | イベントの一覧をカード形式で表示 |
| イベント作成 | イベント作成フォーム |
| イベント詳細 | イベント情報、残り枠数、参加者一覧、参加登録フォームを表示 |
